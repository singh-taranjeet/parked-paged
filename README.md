# Parked Page

This is a simple landing page informing visitors that the website has been relocated.

## New Location

Visit us at: **https://sealedspace.com**

## Features

- Clean, modern landing page design
- Large call-to-action button
- Automatic redirect after 10 seconds
- Fully responsive mobile design
- Beautiful gradient background

## Files

- `index.html` - Main landing page
- `styles.css` - Styling for the page
- `README.md` - This file

## Deployment

Simply host the `index.html` and `styles.css` files on any web server. No backend required.
